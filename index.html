Day 1 – Unity Basics

  1. Install Unity Hub and Unity Editor (2022+)
  2. Create a new 3D project and name it "SummerCampGame"
  3. Open the default scene, add a Plane and Cube (rename to "Player")
  4. Add Rigidbody to Player object
  5. Create and attach PlayerMovement.cs script
  6. Write movement code for WASD / Arrow keys
  7. Attach the Main Camera to follow the Player
  8. Test basic movement in Play mode

Day 2 – Interaction & Modeling

  1. Add jump functionality to PlayerMovement.cs
  2. Create a Collectible prefab with a trigger collider
  3. Write Collectible.cs to destroy object and call GameManager
  4. Create UI with TextMeshPro to show score
  5. Import simple 3D assets or design in Blender
  6. Arrange objects in scene to form level layout

Day 3 – Polish & Publish

  1. Create Main Menu scene with buttons (Start, Quit)
  2. Set up MenuManager.cs script for button actions
  3. Build a win condition and trigger scene change
  4. Design a Win scene with message and Restart button
  5. Build project for WebGL (File > Build Settings > WebGL)
  6. Upload to itch.io or GitHub Pages for publishing
