Day 1 – Unity Setup & Movement

Today, you'll set up your Unity project and create a player that can move around the alien jungle.

  1. Install Unity Hub and Unity Editor 2022.3 LTS.
  2. Create a new 3D project called CosmicJungleGame.
  3. In the Hierarchy, right-click and add a 3D Object → Plane. Rename it AlienGround.
  4. Color the ground purple or green using the Inspector → Mesh Renderer → Materials.
  5. Add a Cube for your player and rename it Player.
  6. Click the Player cube, then go to Inspector → Add Component → Rigidbody.
  7. Tag the Player as Player under the Tag dropdown at the top.
  8. Create a script named PlayerMovement.cs and paste in the movement code.
  9. Attach PlayerMovement.cs to the Player GameObject.
  10. Adjust the camera to follow behind the player, or use a follow script.
  11. Add colorful lights or a spacey skybox to give it a cosmic jungle vibe.
  12. Press Play and walk around using WASD or arrow keys.
