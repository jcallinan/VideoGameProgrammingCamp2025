Day 1 – Unity Setup & Movement

Today, you'll set up your Unity project and create a player that can move around the alien jungle.

  1. Open Unity Hub and create a new 3D project named CosmicJungleGame using Unity 2022.3 LTS.
  2. Create and save a new scene called MainScene.
  3. In the Hierarchy, right-click → 3D Object → Plane. Rename it AlienGround.
  4. Assign it a new material with dark purple or green color. Add it to a new Layer called Ground.
  5. Right-click → 3D Object → Cube. Rename it Player.
  6. Add a Rigidbody and check Freeze Rotation X, Y, Z. Set Drag and Angular Drag to 1.
  7. Tag it Player. Create a LayerMask called Ground and assign it to AlienGround.
  8. Right-click on Player → Create Empty → name it GroundCheck. Move it to Y = -0.6 under the cube.
  9. Create and attach PlayerMovement.cs using the provided script. Fill in the groundCheck and groundMask fields in the Inspector.
  10. Select Main Camera, add CameraFollow.cs and set the target to Player.
  11. Adjust the camera offset to Y = 5, Z = -10. Press Play and test walking and jumping.
