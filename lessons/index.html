Treasure Hunt in the Cosmic Jungle

Theme: You're a space explorer stranded in an alien jungle. Collect the Cosmic Gems to power the portal and escape before time runs out!

Day 1 – Movement & Environment

  1. Install Unity Hub and Unity 2022.3 LTS.
  2. Create a 3D project named CosmicJungleGame.
  3. Add a Plane, rename to AlienGround, color dark purple or green.
  4. Create a Cube for your player, add a Rigidbody, and tag it Player.
  5. Attach PlayerMovement.cs script to your player.
  6. Set the Main Camera to follow the player using a follow script or manually.
  7. Add colored lights and skybox to create jungle atmosphere.

Day 2 – Collectibles & UI

  1. Create a Collectible object using a sphere or crystal, add glowing material.
  2. Add Collectible.cs and attach a magical sparkleSound audio clip.
  3. Create a Canvas with TextMeshPro text elements: one for score, one for timer.
  4. Create a GameManager object, add GameManager.cs, and set totalGems.
  5. Drag Text objects into the GameManager inspector.
  6. Test gameplay: walk around, collect gems, hear sparkle sound, see score update.

Day 3 – Menus, Win, Lose, Polish

  1. Create MenuScene with title, subtitle, and a Start button (MenuManager.cs).
  2. Build WinScene with message and Play Again button.
  3. Create LoseScene with timeout message and retry button.
  4. Add portal object to MainScene, attach WinTrigger.cs.
  5. Add ambient jungle sound to MainScene.
  6. Go to File > Build Settings, add all 4 scenes in order: MenuScene, MainScene, WinScene, LoseScene.
  7. Set MenuScene as your default start scene.
  8. Playtest the entire game: Start > Collect gems > Win or timeout > Retry
