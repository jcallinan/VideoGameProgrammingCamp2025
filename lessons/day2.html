Day 2 – Cosmic Gems, Sparkles, and Score

Today, you'll make glowing gems the player can collect, add sounds, and display score and time.

  1. Create a Sphere in the scene. Scale it to 0.5 and apply a glowing material (enable Emission).
  2. Add a Box Collider, check Is Trigger.
  3. Attach Collectible.cs to the object.
  4. Assign a sparkle sound to the sparkleSound field in the Inspector.
  5. Duplicate the gem and spread several around the terrain.
  6. Create an empty GameObject named GameManager.
  7. Attach GameManager.cs. Set totalGems to the number of gems placed.
  8. Add a Canvas to the scene.
  9. Right-click the Canvas → UI → TextMeshPro - Text. Name it ScoreText.
  10. Create a second TMP Text named TimerText.
  11. Drag ScoreText and TimerText into the GameManager’s inspector fields.
  12. Press Play. Walk into a gem. It should disappear, play sound, and update the score!
