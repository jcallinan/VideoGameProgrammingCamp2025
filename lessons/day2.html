Day 2 – Cosmic Gems and Score

Today you’ll make your collectibles come alive with glowing effects and add a scoring system that shows your progress.

  1. In the Hierarchy, right-click → 3D Object → Sphere. Rename it to CosmicGem and scale it down to 0.5.
  2. Create a new material: In the Project window, right-click → Create → Material → Name it GemMaterial.
  3. Select GemMaterial, then in the Inspector:
    • Click the white color swatch under Albedo and choose a bright color (like cyan or magenta).
    • Scroll down and check the Emission box.
    • Click the color box next to Emission and select a glowing color (brighter version of Albedo).
    • Increase the intensity slider if needed until it glows in Scene view.
  4. Drag GemMaterial onto your CosmicGem sphere in the Hierarchy.
  5. Optional Glow Particles: Select CosmicGem, then right-click → Effects → Particle System.
  6. In the Inspector for the Particle System:
    • Change Start Lifetime to 0.5, Start Speed to 0.5, Start Size to 0.3.
    • Set Max Particles to 50.
  7. Add a Box Collider (Is Trigger = true).
  8. Attach Collectible.cs and assign a sparkle sound to sparkleSound in the Inspector.
  9. Duplicate the gem around the level. Vary their Y-position slightly to make them hover.
  10. In the Hierarchy: Create → UI → Canvas → Add TextMeshPro - Text. Click 'Import TMP Essentials' when that pops up (only have to do this once per project). Name it ScoreText.
  11. Repeat and create TimerText. Style them to match the sci-fi theme (top-left corner, large white font).
  12. Create an empty GameObject → name it GameManager → attach GameManager.cs.
  13. Set totalGems to match the number of gems placed in the scene.
  14. Drag ScoreText and TimerText into the appropriate fields on GameManager.
  15. Press Play. Walk into a Cosmic Gem. It should disappear, play the sparkle, and update your score and timer!
