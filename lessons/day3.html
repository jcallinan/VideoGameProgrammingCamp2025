Day 3 – Menus, Portal, and Game Polish

Today, you'll build the full game flow and finish with a playable camp-ready version!

  1. Create a new scene called MenuScene.
  2. Add a Canvas with a title text and a Start Hunt button.
  3. Create a script MenuManager.cs and attach it to a GameObject in the scene.
  4. Link the Start button's OnClick to call MenuManager.StartGame.
  5. Create WinScene and LoseScene with UI messages and Retry buttons.
  6. In MainScene, add a portal object or arch (can be a glowing cube).
  7. Attach WinTrigger.cs to it, and set its collider to Is Trigger.
  8. Test the game: collect all gems → win scene. Let the timer expire → lose scene.
  9. Go to File → Build Settings and click Add Open Scenes.
  10. Make sure the order is: MenuScene, MainScene, WinScene, LoseScene.
  11. Set MenuScene as the first one, then click Build and Run.
  12. Enjoy your finished Cosmic Jungle Game!
