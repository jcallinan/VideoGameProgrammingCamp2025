Day 3 – Menus, Portal, Win and Lose

Today you’ll build a MenuScene, create Win/Lose scenes, add a glowing portal, and wrap up your game.

  1. Create a new scene called MenuScene. Add a Canvas, Text "Cosmic Jungle", and a button "Start Hunt".
  2. Attach MenuManager.cs to a GameObject. Link the button’s OnClick to StartGame.
  3. Create WinScene: Add Text "You win!" and Button "Play Again" (loads MainScene).
  4. Create LoseScene: Add Text "Time’s up!" and Button "Retry" (loads MainScene).
  5. In MainScene, create a sci-fi portal (using any object, cube, sphere, etc). Call this 'Portal' and add a collider if one doesn't exist, setting (Is Trigger) to True.
  6. Attach WinTrigger.cs to the portal and verify the trigger loads WinScene.
  7. Open Build Settings (File → Build Settings). Add all 4 scenes in order:
    1. MenuScene
    2. MainScene
    3. WinScene
    4. LoseScene
  8. Set MenuScene as the first scene. Click Build and Run or test from Play mode.
  9. Play through: Start → Collect gems → Win. Wait out timer → Lose. You did it!
